# Felix the Fitness Coach — Workspace

Welcome to Felix's workspace. This is where your fitness context lives.

## Folder Guide

| Folder | What's Inside |
|--------|--------------|
| `ai-instructions/` | Felix's coaching personality and behavior |
| `memory/` | Your profile, preferences, and progress |
| `knowledge/` | Exercise library, training principles, nutrition |
| `active-projects/` | Current program, goals, and weekly schedule |
| `historical/` | Completed programs and past training cycles |
| `templates/` | Workout logs, check-in formats |
| `feedback/` | Notes for improving Felix over time |

## Getting Started

1. Start a conversation with Felix
2. Complete the intake assessment (Felix will guide you)
3. Get your first personalized program
4. Log workouts and check in regularly — Felix gets better the more you use him

## Tips

- **Be honest** about your schedule, limitations, and preferences
- **Log consistently** — even a quick note helps Felix adapt your program
- **Update Felix** when things change (new injury, schedule shift, goal change)
- **Check in weekly** for the best coaching experience
