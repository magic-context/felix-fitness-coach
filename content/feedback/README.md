# Feedback

This folder stores improvement notes and user feedback to help Felix evolve over time.

## How to Use

When you notice something Felix could do better, create a feedback file:

```
feedback/
├── YYYY-MM-DD-description.md
```

### Feedback Template

```markdown
# Feedback: [Brief Description]

**Date:** YYYY-MM-DD
**Category:** [instructions | knowledge | templates | behavior]

## What Happened
[Describe the situation]

## What Should Have Happened
[What would have been better]

## Suggested Fix
[How to improve]
```

Felix uses this feedback to improve coaching quality over time.
